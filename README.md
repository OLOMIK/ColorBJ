## Features:
- Drawing
- Importing images (.png, .jpg, .jpeg, .webp)
- Exporting images
- Setting background
- Setting writing color
- Changing brush and eraser size
- Removing background
- Text editing
- Color manipulating

## Technologies:
- Electron.js (for the desktop version)
- Node.js (for the desktop version)
- JavaScript
- HTML
- CSS
