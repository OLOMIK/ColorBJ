## Funkcje:
- rysowanie
- importowanie obrazów (.png,.jpg,.jpeg,.webp)
- eksportowanie obrazów
- ustawianie tła
- ustawianie koloru pisania
- zmiana rozmiaru pędzla i gumki
## Technologie:
- Electron.js (dla wersji desktopowej)
- Node.js (dla wersji desktopowej)
- Javascript
- HTML
- CSS
